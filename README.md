# 🛒 Smart Deals Client

Alhamdulillah, **Smart Deals Client** is a modern bidding marketplace platform built with React, Firebase, and TailwindCSS. It enables buyers and sellers to interact seamlessly — sellers can list products, buyers can place bids, and both parties can manage offers securely. The project emphasizes clean UI, secure API communication, and scalable architecture.

---

## ✨ Key Features

### 🔐 Authentication & Security

- Firebase authentication (sign-in, register, Gmail login).
- Gmail login issue resolved (June 4, 2026).
- Conditional login/logout button in the navbar.
- Private route implementation for protected pages.
- Axios secure instance with interceptors to set headers and handle bad requests.

### 🛍️ Product Management

- Product creation form for sellers.
- Display all products via product card component.
- Show recent products on homepage.
- Seller products table for managing listings.
- Product card details component for individual product view.
- Banner section completed for homepage.

### 💸 Bidding System

- Buyers can place bid offers and send data to backend.
- Buyer bids table displayed in UI.
- Delete operation for buyer bids implemented.
- All bidders list shown in product card details.

### 🖥️ UI & UX Enhancements

- Console logs cleaned up for production readiness.
- Server link added for deployment.
- Dynamic conditional rendering across components.
